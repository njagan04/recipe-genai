This directory stores the FAISS Vector Database and metadata generated by the application.

Because the vector database can be hundreds of megabytes in size, the actual `.index` and `.json` files are ignored by git to keep the repository lightweight.

To generate the database files (`recipes_ivf.index` and `metadata.json`) on your machine:
1. Ensure your cleaned recipe data is placed at `backend/data/processed/recipes_cleaned.json`.
2. Open your terminal in the `backend/` directory and run: `python -m src.retrieval.build_faiss`
